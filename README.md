# Swift-Ride: Real-Time Data Pipeline (Phase 1)

**Project Status:** 🟢 Ingestion Layer Complete

## Description
Swift-Ride is a data engineering project focused on building a scalable, real-time streaming pipeline for ride-sharing analytics. In this first phase, I have successfully developed a Python-based data producer that simulates live ride events and streams them directly into **Azure Event Hubs**.

## Key Achievements So Far
* **Real-time Simulation:** Built a Python producer to generate synthetic ride data.
* **Cloud Integration:** Configured and verified a secure connection between a local development environment (VS Code) and **Microsoft Azure**.
* **Stream Verification:** Successfully validated data ingestion using **Azure Data Explorer**, confirming end-to-end connectivity.

## Next Steps
* Orchestrate data movement using **Azure Data Factory**.
* Perform stream processing and transformations using **Azure Databricks** and **PySpark**.
